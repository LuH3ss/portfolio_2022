Cuál es mi rol?

Tengo como misión hacer una página que se adapte a la mayor cantidad de pantallas (o dispositivos) que haya actualmente. Y no solamente eso. Sino que tambien, tengo que adaptar el diseño a cualquier usuario que visite el sitio.

HTML

HTML es el archivo, dentro de todos los que conforman una pagina, en el cual se organiza la información básica. Pueden ser fotos, video, textos, gifs, etc.

CSS

Lo que se modifica es el aspecto de los items que contienen la información en el archivo HTML. Puede ser el color del texto, bordeados, sombras, paletas de colores y muchos más.

Javascript

Tras todo lo visual (dentro de lo básico), Javascript viene a darle a la pagina la funcionalidad. Los botones, las animaciones y transiciones pueden hacerse con JS.

Sass

Nos ayuda a escribir CSS de una manera más rápida, concreta y sencilla. Gracias a este, los diseños pueden ser mas fáciles de entender dentro de cualquier grupo de trabajo.

Pug

Pug es un motor de plantilla que nos sirve para escribir HTML de una manera mucho mas resumida. Esto nos ahorra tiempo de escritura, lectura, corrección y otras ediciones de codigo.

Bootsrap

Framework que facilita la adaptacion de una página o aplicación web a la pantalla del dispositivo del usuario. Gracias a su sistema de grillas y otros defaults, se ahorra mucho tiempo.
